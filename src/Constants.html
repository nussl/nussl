constants.py¶

A repository containing all of the constants frequently used in this wacky, mixed up source separation stuff.

constants.ALL_WINDOWS = ['hamming', 'rectangular', 'hann', 'blackman', 'triangular']¶

list(str) – list of all available windows in nussl

constants.CHAN_INDEX = 0¶

(int) – Index of the number of channels in an audio signal. Used in AudioSignal Class

constants.DEFAULT_BIT_DEPTH = 16¶

(int) – Default bit depth. 16-bits, CD-quality

constants.DEFAULT_MAX_VAL = 65536¶

(int) – Max value of 16-bit audio file (unsigned)

constants.DEFAULT_SAMPLE_RATE = 44100¶

(int) – Default sample rate. 44.1 kHz, CD-quality

constants.DEFAULT_WIN_LEN_PARAM = 0.04¶

(float) – Default window length. 40ms

constants.EPSILON = 1e-16¶

(float) – epsilon for determining small values

constants.LEN_INDEX = 1¶

(int) – Index of the number of samples in an audio signal. Used in AudioSignal Class

constants.MAX_FREQUENCY = 22050¶

(int) – Maximum frequency representable. 22050 Hz

constants.NUMPY_JSON_KEY = 'py/numpy.ndarray'¶

(str) – key used when turning numpy arrays into json

constants.STFT_CHAN_INDEX = 2¶

(int) Index of the number of channels in a time-frequency representation. Used in AudioSignal Class and in MaskBase Class.

constants.STFT_LEN_INDEX = 1¶

(int) Index of the number of time (horizontal) hops in a time-frequency representation. Used in AudioSignal Class and in MaskBase Class.

constants.STFT_VERT_INDEX = 0¶

(int) Index of the number of frequency (vertical) values in a time-frequency representation. Used in AudioSignal Class and in MaskBase Class.

constants.WINDOW_BLACKMAN = 'blackman'¶

(str) – Name for calling Blackman window. ‘blackman’

constants.WINDOW_DEFAULT = 'hamming'¶

(str) – Default window, Hamming.

constants.WINDOW_HAMMING = 'hamming'¶

(str) – Name for calling Hamming window. ‘hamming’

constants.WINDOW_HANN = 'hann'¶

(str) – Name for calling Hann window. ‘hann’

constants.WINDOW_RECTANGULAR = 'rectangular'¶

(str) – Name for calling Rectangular window. ‘rectangular’

constants.WINDOW_TRIANGULAR = 'triangular'¶

(str) – Name for calling Triangular window. ‘triangular’
